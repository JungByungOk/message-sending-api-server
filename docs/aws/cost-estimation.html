비용 절감 팁
1. DynamoDB Provisioned 모드 — 트래픽 패턴이 예측 가능하면 On-Demand → Provisioned 전환으로 ~40% 절감
2. Lambda ARM64 — Graviton2 프로세서 사용 시 ~20% 비용 절감 (CDK에서 architecture: lambda.Architecture.ARM_64)
3. SES 전용 IP 불필요 — 월 10만 통 이하에서는 공유 IP로 충분
4. CloudWatch 로그 보존 — 로그 보존 기간을 30일로 설정하여 저장 비용 절감
5. 리소스 정리 — 테스트 후 cdk destroy로 전체 삭제하면 과금 즉시 중단
